## Minimum scenario (from teacher prompt)
Keyboard-driven drawing program that supports:
- **begin** (`b`) create a new polyline (up to 100)
- **delete** (`d`) delete closest point
- **move** (`m`) drag closest point
- **refresh** (`r`) erase and redraw all polylines
- **quit** (`q`) exit (web version: clear/reset or disable interaction)
