# Scenarios

## Scenario A: Drawing a house outline (from teacher prompt)
1. User presses `B` to begin a new polyline.
2. User clicks multiple points to draw walls/roof.
3. User presses `F` to finish the polyline.
4. User presses `B` again to draw another polyline (door/window).

## Scenario B: Fixing the roof corner
1. User presses `M` for move mode.
2. User moves cursor near a vertex.
3. User drags it to adjust roof angle.

## Scenario C: Removing an incorrect point
1. User presses `D` for delete mode.
2. User clicks near the wrong vertex.
3. System removes the point and reconnects line segments.
