# Challenges & Confusions

1) Meaning of “Quit” in a web app  
- Prompt says exit program; in web we can clear/reset state or disable input and show message.

2) Refresh vs redraw  
- The app redraws frequently; still implement a dedicated `refresh` action for compliance.

3) Undo behavior  
- Prompt doesn’t require undo, but it improves usability; must be documented clearly if included.
